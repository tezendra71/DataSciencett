# Energy Forecasting

This is an energy forecasting project that builds models to predict power consumption based on historical data and time-based features.

## Project Structure

- `data/`: Contains the training and testing datasets.
- `src/`: The python package containing the data pipeline, preprocessing, and model training scripts.
- `models/`: Trained model binaries.
- `notebooks/`: Jupyter notebooks for data exploration.

## Setup

1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

2. Run the pipeline:
   ```bash
   python src/pipeline.py
   ```
